# Laravel JWT Authentication API

This is a complete Laravel 8.x project that demonstrates user registration, password reset functionality, JWT authentication, and custom database notifications.

## Features

- **JWT Authentication**: Secure the API using JSON Web Tokens (JWT).
- **User Registration**: Register a new user and receive a JWT token for authentication.
- **Password Reset**: Forgot password functionality with email notifications and database notifications.
- **Custom Notifications**: Notifications sent when certain actions occur (e.g., password reset).
- **Change Logging**: Logs critical changes such as user registration and password reset events.
